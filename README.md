
# SasaMum

**Because Every Mother Deserves Smart Care**

SasaMum is a smart maternal health companion designed to support mothers through pregnancy, postpartum, and baby care. With AI-powered tools, personalized insights, and emergency readiness features, SasaMum ensures every mother receives holistic and intelligent care.
